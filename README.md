CLI-утилита для анализа, построения и визуализации графа зависимостей программных пакетов. Проект демонстрирует:

работу с конфигурационными файлами YAML

загрузку и парсинг индекса пакетов из реального репозитория

построение графа зависимостей

поиск циклических зависимостей

формирование порядка загрузки пакетов

визуализацию зависимостей в формате Graphviz DOT и ASCII-дерева

поддержку тестового режима на основе локального файла-графа.

Все параметры задаются в файле config.yaml
Пример:

package_name: "A"

repo_url: "test_repo.txt"

repo_mode: "test"

ascii_tree: true

filter_substring: ""

load_order: true

package_name(string) - Корневой пакет, с которого начинается анализ

repo_url(string) - URL репозитория Alpine или путь к тестовому файлу

repo_mode(string) - real — работа с Alpine репозиторием, test — работа с локальным графом

ascii_tree(bool) - Если true, выводит зависимости в виде ASCII-дерева

filter_substring(string) - Исключает из анализа пакеты, содержащие подстроку

load_order(bool) - Если true, выводит порядок загрузки зависимостей
